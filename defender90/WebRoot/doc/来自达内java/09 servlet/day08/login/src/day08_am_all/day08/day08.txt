		绑订相关数据。比如:
			session.setAttribute("user",user);

		能访问的地址,比如，main.jsp),添加验证代码，
		比如:
			Object obj = session.getAttribute("user");
			if(obj == null){
				response.sendRedirect("login.jsp");
			}
